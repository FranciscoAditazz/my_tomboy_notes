Class

A basic class consists only of the class keyword, the name of the class, and the class from which the new class inherits in parentheses, like so: class NewClass(object): # This class will inherit from the object class # Class magic here pass pass doesn't do anything, but it's useful as a placeholder in areas of your code where Python expects an expression. Example class Animal(object): pass __init__() This function is required for classes, and it's used to initialize the objects it creates. The first argument __init__() gets is used to refer to the instance object, and by convention, that argument is called self, that refers to the object being created. You can think of __init__() as the function that "boots up" each object the class creates. If you add additional arguments—for instance, a name and age for your animal—setting each of those equal to self.name and self.age in the body of __init__() will make it so that when you create an instance object of your Animal class, you need to give each instance a name and an age, and those will be associated with the particular instance you create. Class Scope Not all variables are accessible to all parts of a Python program at all times. When dealing with classes, you can have variables that are available everywhere (global variables), variables that are only available to members of a certain class (member variables), and variables that are only available to particular instances of a class (instance variables).
