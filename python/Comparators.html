Comparators

The simplest aspect of control flow: comparators. There are six: 01. Equal to (==) 02. Not equal to (!=) 03. Less than (<) 04. Less than or equal to (<=) 05. Greater than (>) 06. Greater than or equal to (>=) Note that == compares whether two things are equal, and = assigns a value to a variable. Boolean operators Compare statements and result in boolean values. There are three boolean operators: 01. and, which checks if both the statements are True; 02. or, which checks if at least one of the statements is True; 03. not, which gives the opposite of the statement. Example: bool_one = 2**3 == 108 % 100 or 'Cleese' == 'King Arthur' bool_two = True or False bool_three = 100**0.5 >= 50 or False bool_four = True or True bool_five = 1**100 == 100**1 or 3 * 2 * 1 != 3 + 2 + 1 print "1=%s, 2=%s, 3=%s, 4=%s, 5=%s" % (bool_one, bool_two, bool_three, bool_four, bool_five)
