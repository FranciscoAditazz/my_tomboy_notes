Bitwise Operations

Bitwise operations are operations that directly manipulate bits. In all computers, numbers are represented with bits, a series of zeros and ones. print 5 >> 1 # Right Shift prints 2 0b101 >> 1 = 0b10 print 5 << 1 # Left Shift prints 10 0b101 << 1 = 0b1010 print 8 & 5 # Bitwise AND prints 0 0b1000 & 0b101 = 0b0 print 9 | 4 # Bitwise OR prints 13 0b1001 | 0b100 = 0b1101 print 12 ^ 42 # Bitwise XOR prints 38 0b1100 ^ 0b1100 = 0b100110 print ~88 # Bitwise NOT prints -89 ~0b1011000 = -0b1011001 bin() bin() takes an integer as input and returns the binary representation of that integer in a string. (Keep in mind that after using the bin function, you can no longer operate on the value like a number.) Using the XOR (^) operator is very useful for flipping bits. Using ^ on a bit with the number one will return a result where that bit is flipped. Example: a = 0b11101110 # 0b11101110 mask = 0b11111111 # 0b11111111 desired = a ^ mask print bin(desired) # 0b 10001 Slip and Slide The following function is an example of how the nth bit can be flipped (with the ones bit being the first bit) def flip_bit(number, n): # 0b11011 <= Original number mask = 0b1 << n-1 print bin(mask) # 0b 1000 <= Mask result = number ^ mask # 0b10011 <= Result return bin(result) print flip_bit(0b11011, 4)
