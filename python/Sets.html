Sets

Sets are collections of unique but unordered items. It is possible to convert certain iterables to a set. Example: new_set = {1, 2, 3, 4, 4, 4,'A', 'B', 'B', 'C'} print new_set # prints {'A', 1, 'C', 3, 4, 2, 'B'} dup_list = [1,1,2,2,2,3,4,55,5,5,6,7,8,8] set_from_list = set(dup_list) print set_from_list # prints {1, 2, 3, 4, 5, 6, 7, 8, 55}
