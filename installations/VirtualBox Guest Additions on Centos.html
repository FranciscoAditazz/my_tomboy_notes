VirtualBox Guest Additions on Centos

Para poder corregir el error que marca cuando se intenta correr el vBoxGuestAdditions se hace lo siguiente: a) Se carga el repositorio y se instalan las librerías que se indican: rpm -Uvh http://dl.fedoraproject.org/pub/epel/7/x86_64/e/epel-release-X-X.noarch.rpm yum install htop yum install dkms b) Dependiendo lo que indique el error en el log: se instala la recomendación del kernel-devel: cat /var/log/vboxadd-install.log yum install kernel-devel-3.10.0-123.13.2.el7.x86_64 c) Se corrige el link virtual del archivo build: cd /lib/modules/3.10.0-123.13.2.el7.x86_64/ rm build ln -s /usr/src/kernels/3.10.0-229.4.2.el7.x86_64/ build d) Se corre el Guest Additions ./VBoxLinuxAdditions.run
