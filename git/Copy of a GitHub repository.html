Copy of a GitHub repository

The steps to copy a repository from another GitHub repository 1) on a linux shell clone the repository to be copied: $ git clone https://github.com/xfrarod/jpetstore 2) Change to the directory $ cd jpetstore 3) Reinitialize the git repository $ git init 4) On GitHub create a blank repository with the same name jpetstore (https://github.com/<Kim's repo>/jpetstore) 5) Change the origin repo $ git remote set-url origin https://github.com/<Kim's repo>/jpetstore 6) Push the information to the new repo $ git push -u origin master
