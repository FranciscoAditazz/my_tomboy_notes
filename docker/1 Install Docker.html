1 Install Docker

1) yum update 2) wget http://ftp.osuosl.org/pub/fedora-epel/6/i386/epel-release-6-8.noarch.rpm 3) rpm -Uvh epel-release-6-8.noarch.rpm 4) yum update 5) yum install docker-io By using automated script: • For curl command: $ sudo curl -sSL https://get.docker.io/ | sh • For wget command: $ sudo wget -qO- https://get.docker.io/ | sh
