vagrant init

This will place a Vagrantfile in your current directory. $ vagrant init A `Vagrantfile` has been placed in this directory. You are now ready to `vagrant up` your first virtual environment! Please read the comments in the Vagrantfile as well as documentation on `vagrantup.com` for more information on using Vagrant. You can also run vagrant init in a pre-existing directory to set up Vagrant for an existing project.
