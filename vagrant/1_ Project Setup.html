1. Project Setup

1. Mark the root directory of your project. A lot of the configuration of Vagrant is relative to this root directory. 2. Describe the kind of machine and resources you need to run your project, as well as what software to install and how you want to access it. Vagrant has a built-in command for initializing a directory for usage with Vagrant: vagrant init. For the purpose of this getting started guide, please follow along in your terminal: $ mkdir vagrant_getting_started $ cd vagrant_getting_started $ vagrant init
