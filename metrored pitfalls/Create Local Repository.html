Create Local Repository

1) Copy the rpm files to a directory where the local repository will be located 2) Create rhel.repo file in the /etc/yum.repos.d directory with the following contents: [rhel-local] name=Red Hat Enterprise Linux baseurl=file:///mnt/local_repo enabled=1 gpgcheck=0 3) Give the create repo command and clear cache createrepo --database /mnt/local_repo yum clean all Note: if createrepo is not installed, install it manually with the following command: rpm -Uvh python-deltarpm-3.5-0.5.20090913git.el6.x86_64.rpm deltarpm-3.5-0.5.20090913git.el6.x86_64.rpm createrepo-0.9.8-4.el6.noarch.rpm
