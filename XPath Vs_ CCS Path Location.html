XPath Vs. CCS Path Location

CCS location is more elegant way than XPath <html> <head></head> <body> <form id="loginForm" style="background-color:transparent;"> <input id="u1" type="text" name="username" style="background-color: rgb(255, 255, 255);"> <input type="password" name="password"> <input type="submit" value="Login" name="continue"> </form> </body> </html> Example 1: Locate complete html in XPath : /html CSS Path: html Example 2: Locate form XPath : /html/body/form CSS Path: html>body>form Example 3: Locate id=u1 in XPath : //input[@id='u1'] CSS Path: input[id=u1] or input#u1 or #u1 Example 4: Locate submit button in dynamic id XPath : //input[starts-with(@id,'submit')] CSS Path: input[id ^=submit] Example 5: Locate qty field in dynamic id XPath : //input[contains(@id,'count')] CSS Path: input[id *=count]
