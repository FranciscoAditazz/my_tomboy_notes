How to Create a Web App in 5 min

1) Create a directory for the project mkdir code cd code 2) Create the container and artifacts necessary... and the baseline widgets which are call gems that get the server up and running rails new app1 3) Create the M - data access layer, C - the business layer and V - the presentation layer (MVC). rails generate scaffold post title text:text 4) Create the database base on our model rake db:migrate 5) Start the web server rails s For publishing the application on horuku 6) Edit the Gemfile (which contains all the widgets that the aplications to use), and do the following: replace: gem 'sqllite3' by: group :production do gem 'pg' end group :development, :test do gem 'sqllite3' end 7) since horuku only supports postgresql (pg) database, we have to install the gem gem install pg
