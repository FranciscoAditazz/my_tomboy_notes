Import

It's possible to import only certain variables or functions from a given module. Pulling in just a single function from a module is called a function import, and it's done with the from keyword: from module import function Example: from math import sqrt Universal Imports What if we still want all of the variables and functions in a module but don't want to have to constantly type math.? Universal import can handle this for you: from module import * The next code will show you everything available in the math module: import math # Imports the math module everything = dir(math) # Sets everything to a list of things from math print everything # Prints 'em all!
