Selenium Remote Control Server (RC)

A Client (any program in java, perl, php, python, ruby) sends commands (open, clickAndWait, type, echo, etc) to the Server (RC) and therefore the server communicates to the Browser (FireFox, Chrome, IExplorer, etc) through the Selenium Core (SC) which is basically using javascript. References:
