Selenese commands

Those are the commands that can be used on Selenium. Three categories of commands exist: Action
  • Cick
  • Select
Accessors (Help to determine the state of our page)
  • Store Test
Assertion (Verify the state of Web page & Confirmation) Each command has three fields
  • Command (What)
  • Target (Where, how to identify the element on the Web Page)
    • By ID
    • By Name
    • By link
    • By XPath
    • By CSS Path
    • By DOM
  • Value (optional)
