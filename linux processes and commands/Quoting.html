Quoting

Double quote
  • Substitutes the value of variables and commands
Example: echo “Username is $USER" The above will print what the current username Single quote
  • This preserves the literal meaning of each character of a given string
  • This will turn off the (special) meanings of all characters
Example: echo ‘Username is $USER’ The above example will print/echo the literal Username of $USER xfrarod@Satellite-L955 /var/log $ echo "the name is $USER" the name is xfrarod xfrarod@Satellite-L955 /var/log $ echo 'the name is $USER' the name is $USER
