Truquear Centos para que se vea como RHEL

Se modifica el archivo /etc/redhat-release, ejemplo: [root@dash01 ~]# cat /etc/redhat-release Red Hat Enterprise Linux Server release 6.6 [root@dash01 ~]# cat /etc/redhat-release.centos CentOS release 6.6 (Final) [root@dash01 ~]# ll /etc/*release -rw-r--r--. 1 root root 44 Jun 5 08:25 /etc/centos-release -rw-r--r--. 1 root root 152 May 16 04:47 /etc/lsb-release lrwxrwxrwx. 1 root root 14 May 15 23:43 /etc/redhat-release -> centos-release lrwxrwxrwx. 1 root root 14 May 15 23:43 /etc/system-release -> centos-release Para validar el release se utiliza el comando: > lsb_release -a LSB Version: :base-4.0-amd64:base-4.0-noarch:core-4.0-amd64:core-4.0-noarch:graphics-4.0-amd64:graphics-4.0-noarch:printing-4.0-amd64:printing-4.0-noarch Distributor ID: RedHatEnterpriseServer Description: Red Hat Enterprise Linux Server release 6.6 Release: 6.6 Codename: n/a
