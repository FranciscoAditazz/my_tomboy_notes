Port Troubleshooting (validar puerto)

Para checar que puertos existen abiertos actualmente sudo lsof -n -i -P Para saber si el puerto esta escuchando netstat -an | grep -i <Port Number> Ejemplo: netstat -an | grep -i 60000 Para conocer los recursos TCPIP en la m√°quina pctt -r TCPIP Resources... +--------------------------------------------+ | HostName : NMU-GST-PERF-DM | | HostAddress: NMU-GST-PERF-DM | | IPAddress : 10.10.3.196 | +--------------------------------------------+ Para saber si el puerto no esta bloqueado por iptables iptables -L Para poder abrir algun puerto en iptables se usan los sig. comandos: sudo iptables -I INPUT -p tcp --dport 22 -j ACCEPT sudo /sbin/service iptables save
