Mount New Disk

When a new disk is created on VirtualBox, it is needed to make the OS to recognize it: The steps for mounting the new disk are a) First add a new hard disk. Go to the virtual media manager and add a disk to the Virtual Machine b) Now after booting your guest you need to do some linux magic for the OS to recognize the hard disk c) The 1st step is to determine the device name of the newly created disk.
